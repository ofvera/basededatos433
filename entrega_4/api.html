Documentación API Entrega 4

Proposito:

El proposito de esta Web API es exponer en formato JSON información sobre una filtración de mensajes enviados por muchas celebridades del mundo de la música, de una forma accesible y eficiente.

Funcionalidades:

Busqueda por usuario

url: bd11.ing.puc.cl/user/<uid>

Busqueda por mensaje

url: bd11.ing.puc.cl/messages/<mid>

Busqueda por usuarios

url: bd11.ing.puc.cl/users/<uid1>/<uid2>

Busqueda por fechas

url: bd11.ing.puc.cl/filterdates/<before>/<after>/<uid>

Busqueda por usuario

url: bd11.ing.puc.cl/user/<uid>
